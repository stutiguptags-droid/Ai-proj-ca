# AI Cooking Assistant 🍳

A smart recipe recommendation tool that suggests what you can cook based on ingredients you already have.

## Features
- Fast recipe suggestions
- Shows time, ingredients, and instructions
- Simple, clean UI
- Extensible for future AI features

## Tech
- Python
- Streamlit

## How to Run
pip install -r requirements.txt
streamlit run app.py


## Future Enhancements
- NLP for better ingredient understanding
- Cloud deployment
- Personalized recommendations
